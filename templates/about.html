🔍 About SignVerse

SignVerse is a real-time American Sign Language (ASL) detection system designed to bridge communication gaps between the hearing and speech-impaired communities. This project was built as part of our college innovation drive 🧑‍🎓❤️.

🚀 Key Features

  • Real-time ASL detection using webcam
  • Supports all alphabets (A-Z)
  • Live text output
  • Text-to-speech functionality (Coming Soon...)
  • User-friendly interface

🧠 Tech Stack

  • Frontend: HTML, Tailwind CSS, JavaScript
  • Backend: Flask (Python)
  • ML Model: Random Forest Classifier (.p file)
  • Vision: OpenCV + MediaPipe
  • Speech: pyttsx3 (Text to Speech)

⚙️ How It Works

SignVerse captures hand gestures using your webcam, extracts keypoints using MediaPipe, and feeds them into a trained ML model to detect ASL characters.

The predicted letters are displayed in real-time and can be converted into voice using text-to-speech — making the communication two-way and interactive.
